 ## Installation

## Install my-project with npm

```bash
  npm install my-project
  cd my-project
  npm install -D tailwindcss postcss autoprefixer
  npx tailwindcss init -p
  npm install react-router-dom
  tailwind.config.js:
      content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
     ],
     index.html:
   @tailwind base;
    @tailwind components;
    @tailwind utilities;

    npm install sweetalert --save

    npm run dev

```
